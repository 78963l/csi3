CSI
디지털아이디어의 검색방식을 가진 프로젝트 매니징 툴입니다.
빠른속도, 교육의 최소화가 컨셉 입니다.
프로젝트를 진행하면서 기능 및 형태가 수정될 수 있습니다.
Search

검색영역에서는 검색어, 상태, 정렬방식을 설정할 수 있습니다.
검색어1 검색어2 ... : 검색어를 연속을 입력하면, 각 검색어가 포함된 결과가 검색됩니다.
팀코드 아티스트명 : 아티스트명으로 검색합니다.
model,mm,layout,ani,fx,mg,fursim,sim,crowd,light,comp,matte,env : 각팀의 테스크를 검색합니다.
all,올,전체 : 전체가 검색됩니다.(샷+에셋)
shot, 샷, 전샷, 전체샷 : 샷만 검색합니다.
asset, assets, 에셋 : 에셋리스트를 보여줍니다.
1208 : 당일 마감되어야 하는 테스크를 검색합니다.
2016-11-24 : 해당 날짜의 데일리 mov를 검색합니다.
전권 : 영화는 보통 1~8권으로 이루어져 있습니다. 전권,1권..8권도 검색됩니다.
#태그명 : #태그명으로 태그검색이 가능합니다.
2D,3D : 각각 2D, 3D샷이 검색됩니다.
texture, 텍스쳐 : 에셋리스트를 보여줍니다.
00:12:23:12 : 키코드를 검색합니다.(스캔키코드, 작업키코드)
task:model   1208 : 각팀의 테스크별 마감일자를 검색합니다. ((ex) task:matte 1208, task:ani,light 1208)
task:model   2017-12-08 : 각팀의 테스크별 데일리를 검색합니다. ((ex) task:matte 2017-12-08, task:ani,light 2017-12-08)
Deadline / Tag

마감일, 프로젝트 태그모습입니다. 클릭하면 관련 아이템이 검색됩니다.
빨강색이면 이번주, 주황색이면 다음주 관련 항목을 나타냅니다.
Asset Tags

에셋의 타입은 char(케릭터), env(환경), prop(소품), comp(합성에셋), plant(식물), vehicle(탈것) 이 있습니다.
에셋의 형태는 assembly와 component가 있습니다.
assembly는 2개이상의 집합으로 이루어진 에셋입니다. assembly는 assembly를 포함할 수 있습니다.
component는 더이상 나눌수 없는 최소단위의 에셋입니다.
어셈블리 태그 앞에는 링크아이콘이 붙어 있습니다. 링크아이콘을 누르면 이 어셈블리가 포함하고 있는 모든 에셋이 검색됩니다.
에셋태그를 수정할 때는 편집아이콘을 누르고 assettags 항목을 수정해주세요.
규칙1 : 부모 에셋의 이름을 넣으면 에셋이 연결됩니다.
규칙2 : 자신이 assembly 라면 자신의 이름도 assettags에 들어가야 합니다.
Basic

기본 정보입니다. 롤넘버(BO270), 샷이름(S036_0010), 타입(org)를 보여줍니다.
샷타입(2D), 렌즈디스토션사이즈(2112 x 1188), 2D,3D마감일과 핸들IN/OUT을 보여줍니다.
선택항목, 수정, Plate폴더열기, 샷 로그를 볼 수 있는 아이콘이 하단에 배치되어 있습니다.
썸네일을 클릭하면 영상이 플레이됩니다.
Send Mail

메일프로그램과 웹브라우저 설정이 잘 되어있다면 샷 이름을 클릭시 프로젝트명과 샷이름이 적힌 메일창이 열립니다.
이 기능이 작동되기 위해서는 여러분의 브라우져에 메일프로그램 등록이 필요합니다.
firefox : 편집 > 환경변수 > 프로그램탭 > mailto
위 항목을 Thunderbird 메일로 바꾸어주세요.
Thunderbird는 /usr/bin/thunderbird 경로에 존재합니다.
Neutral Grade

플레이트를 CG작업에 유리한 컬러로 바꿀 때 사용합니다.
참고 : 브라우저 버전이 낮다면 청색 라인만 표시됩니다.
플레이트의 컬러를 바꾸면 당연히 Color작업과 관련된 팀을 위해서 표시가 되어야합니다.
CSI는 썸네일이미지 주변에 뉴트럴그레이드 표시가 생성됩니다.
이 기능은 뉴크와 연동되며 뉴크상 단축키는 다음과 같습니다. 컬러 저장하기:alt+[, 컬러 불러오기 alt+]
개념은 OCIO의 cc파일이 DB에 저장되고 아티스트간 공유하는 개념입니다.
Status

작업내용이 각 Task별로 추적되는 상태박스 입니다.
팀명, ▶ 표시, 아티스트명, 마감일이 적혀있습니다.
▶ 표시를 클릭하면 최근 등록된 mov가 재생됩니다.
아티스트명을 클릭하면 작업폴더로 바로 연결되는 WFS(Web File System)이 실행됩니다.
Note

작업에 필요한 기본정보, 레퍼런스, 소스, 수정내용, 태그리스트 순으로 적혀있습니다.
일정설명
편집아이콘을 누르면 각 Task에 일정설정부분이 보입니다.

사람아이콘 : 아티스트명을 적습니다.
S : 작업시작일 입니다.
? : 작업예상일 입니다.
1 : 1차마감일 입니다. 1차 렌더이미지, 테스트데이터..등등 최종 마감전 팀간 교환할 데이터의 마감일이 존재할 때 사용합니다.
2 : 2차마감일 입니다. PM과 약속하는 최종 마감일이며 PM입력기와 연동됩니다.
퀵타임 : 작업 Mov파일 경로를 입력하는 곳입니다.
