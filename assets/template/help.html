Deadline / Tag

마감일, 프로젝트 태그모습입니다. 클릭하면 해당 마감샷만 검색됩니다. 빨강색이면 이번주를, 주황색이면 다음주를 의미합니다.

Asset Tags

에셋의 타입은 char(케릭터) env(환경) prop(소품) comp(합성에셋) plant(식물) vehicle(탈것) 이 있습니다.

에셋의 형태는 여러개의 에셋이 조합되어 하나의 에셋이 되는 assembly와 각 요소를 상징하는 component가 있습니다.

assembly는 2개 이상의 에셋의 집합으로 이루어진 에셋입니다. assembly는 assembly를 포함할 수 있습니다. component는 더이상 나눌수 없는 최소단위의 에셋입니다.

어셈블리 태그 앞에는 링크아이콘이 표시되어 있습니다. 링크 아이콘을 클릭하면 이 assembly와 연결된 모든 에셋이 출력됩니다.

에셋태그를 수정할 때는 편집아이콘을 누르고 assettags 항목을 수정해주세요.

규칙1 : 자식 에셋태그에 부모 에셋의 이름을 넣으면 에셋이 자동으로 연결됩니다.

규칙2 : 자신이 assembly 라면 자신의 이름도 assettags에 들어가야 합니다.

Send Mail

여러분의 웹브라우저가 메일프로그램과 잘 연결되어 있다면, 샷 이름 클릭시 프로젝트명과 샷이름이 적힌 메일창이 열립니다.

이 기능이 작동되기 위해서는 여러분의 웹 브라우저에 메일프로그램 등록이 필요합니다.

Firefox : 편집 > 환경변수 > 프로그램 > mailto

에서 메일 클라이언트를 Thunderbird 메일로 바꾸어주세요.

리눅스에서 Thunderbird의 위치는 /usr/bin/thunderbird 경로입니다.

Neutral Grade(digitalidea)

CSI는 뉴트럴그레이드 파이프라인을 처리할 수 있습니다.

뉴트럴그레이드는 원본 플레이트를 CG작업에 유리한 컬러로 바꿀 때 사용합니다.

플레이트의 컬러를 바꾸게 되면 Color 작업과 관련된 모든 팀을 위해 눈에 띄어야 합니다.

뉴트럴그레이드가 활성화되면 CSI는 썸네일이미지 주변에 뉴트럴그레이드 표시가 활성화 됩니다.

또한 이 기능은 디지털아이디어 내에서 뉴크와 연동되며 뉴크상 단축키는 다음과 같습니다.

컬러 저장하기 : Alt + [, 컬러 불러오기 Alt + ]

큰 개념은 OpenColorIO의 .cc 파일 또는 .ccc 파일을 DB에 저장하고 아티스트간 공유하는 개념입니다.

Status

작업내용이 각 Task별로 추적되는 상태박스 입니다.

팀명 ▶ 표시 아티스트명 마감일 이 적혀있습니다.

▶ 표시를 클릭하면 최근 등록된 mov가 재생됩니다.

작업에 필요한 기본정보, 레퍼런스, 소스, 수정내용, 태그리스트 순으로 적혀있습니다.
